CHAPTER 1 – INTRODUCTION

1.1 Background of the Study

In the modern era, lifestyle-related diseases such as obesity and hypertension are rising. A primary driver is nutritional illiteracy—the inability to gauage meal density accurately.

1.2 Problem Statement

Manual data entry leads to user fatigue. There is a need for a low-friction, AI-driven system to decode meal profiles from a single photograph.

1.3 Objectives

Automate tracking via a Neural Food Scanner, implement biometrics, and use a premium 3D UI (Whobee AI) for engagement.

2.2 Existing System

Standard apps like MyFitnessPal require heavy manual input and lack predictive health feedback for metabolic baselines.
